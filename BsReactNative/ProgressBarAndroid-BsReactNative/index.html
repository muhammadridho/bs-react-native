Android-only component https://facebook.github.io/react-native/docs/progressbarandroid used to indicate that the app is loading or there is some activity in the app.

Example of use

Horizontal

let component = ReasonReact.statelessComponent("ExampleHorizontal");

let make = _children => {
  ...component,
  render: _children =>
    <View>
      <ProgressBarAndroid styleAttr=`Horizontal />
      <ProgressBarAndroid styleAttr=`Horizontal color="#2196F3" />
      <ProgressBarAndroid styleAttr=`Horizontal progress=0.5 />
    </View>,
};

Inverse

let component = ReasonReact.statelessComponent("ExampleInverse");

let make = _children => {
  ...component,
  render: _children =>
    <View>
      <ProgressBarAndroid styleAttr=`Inverse />
      <ProgressBarAndroid styleAttr=`Inverse color="#2196F3" />
    </View>,
};
